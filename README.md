# ProjetoAgrinho2026 
Este site foi desenvilvido como parte do Projeto Agrinho,com o objetivo de apresentar a importância e as caracteristicas do agro no sudoeste do Paraná,destacando a agricultura, a sustentabilidade, a preservação ambiental e as tecnologias utilizadas no campo deforma criativa e interativa. 
O projeto tambem serve como prática de programação web utilizando linguagem HTML,CSS e JavaScript. 

# Funcionalidades 
- Layout moderno e responsivél; 
- Galeria de imagens do agro; 
- Cards informativos; 
- Navegação suave; 
- Menu de acessibilidade; 
- Efeitos visuais; 
- Quiz interativo.
